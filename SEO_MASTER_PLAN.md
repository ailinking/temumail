## 2. Technical SEO Roadmap

### Site Architecture
*   **Flat Structure**: Homepage > Blog > Article.
*   **URL Structure**: \https://temumail.com/blog/article-slug\
*   **Internal Linking**: Breadcrumbs on blog posts, "Create Email" CTA in nav and footer.

### Optimization Checklist
- [ ] **Core Web Vitals**: Ensure LCP < 2.5s, FID < 100ms, CLS < 0.1.
- [ ] **Mobile Responsiveness**: Tailwind CSS ensures this, verify touch targets.
- [ ] **SSL/HTTPS**: Handled by Vercel.
- [ ] **XML Sitemap**: Auto-generated via Next.js.
- [ ] **Robots.txt**: Auto-generated via Next.js.
- [ ] **Canonical Tags**: Self-referencing on all pages.

### Schema Markup (JSON-LD)
*   **WebSite**: Global site identity.
*   **SoftwareApplication**: For the webapp itself.
*   **Article**: For blog posts.
*   **FAQPage**: For the FAQ section.
