# TemuMail - Temporary Email Service

TemuMail is a secure, anonymous, and temporary email service built with Next.js and Tailwind CSS, powered by the Mail.tm API.

## Features

- **Instant Email Creation**: Generate random email addresses instantly.
- **Real-time Inbox**: Auto-refreshing inbox for new messages.
- **Privacy Focused**: No personal data required.
- **Secure**: Implements security headers and safe HTML rendering.
- **Responsive UI**: Works great on desktop and mobile.
